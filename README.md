# Anonymess
Anonymess secure messaging app

## Getting Started

You will need Python3 with flask.

### Prerequisites

Python3
Python-Flask
flask_socketio

### Installing
```
1. Install Python3 any version will work.
2. pip install python-flask
3. pip install flask_socketio
4. pip install emit
5. locate directory, python anonymess.py
```
